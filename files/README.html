OpenStack QEMU Images

These are x86_64 QEMU images, intended to be used with OpenStack. These images use UEFI booting instead of classic BIOS booting.

  1. Download <IMAGE>.qcow2.xz and <IMAGE>.qcow2.xz.asc.
  2. Verify signature: 
    gpg --keyserver hkp://keyserver.ubuntu.com --recv-keys 21412672518D8B2D1862EFEF5CD5D12AA0877B49
gpg --verify <IMAGE>.qcow2.xz.asc <IMAGE>.qcow2.xz
  3. Uncompress image: 
     xz -T0 -d <IMAGE>.qcow2.xz
  4. In OpenStack controller: 
    image create  <IMAGE> \
   --file <IMAGE>.qcow2 \
   --disk-format qcow2 --container-format bare \
   --property hw_firmware_type=uefi \
   --property architecture=amd64 --public
    Note: The images use UEFI boot, not BIOS boot.The hw_firmware_type setting is important therefore!
