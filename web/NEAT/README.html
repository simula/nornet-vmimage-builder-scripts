NEAT VM Images

These are VirtualBox OVA images for the NEAT Development Environment. They are a showcase for NEAT, and of course for the RSPLIB implementation. This directory contains the demo setup for current Ubuntu LTS and Fedora versions. Just select the operation system variant of your choice.

See https://neat.nntb.no/ for a detailed description of NEAT!
