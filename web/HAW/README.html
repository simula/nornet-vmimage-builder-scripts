HAW Computer Networks (Rechnernetze) VM Images

These are VirtualBox OVA images for the HAW Computer Networks (Rechnernetze) exercises. This directory contains the VMs for current Ubuntu LTS and Fedora versions. Just select the operation system variant of your choice. There are two VM variants available:

  • Desktop: Full KDE Desktop installation, with development tools, Firefox preconfigured with uBlock Origin and NoScript, LibreOffice, Wireshark, Gimp, LaTeX, Kile, HiPerConTracer, NetPerfMeter, SubNetCalc, etc.
  • Router: Installation without GUI, for using as a router.
